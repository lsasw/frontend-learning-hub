1变量与数据类型

Python 是动态类型语言，变量无需声明类型，赋值即定义。内置 8 种核心数据类型覆盖了绝大多数场景。

1.1 八大数据类型一览

🔢int 整数

x = 42

任意精度，不会溢出

💧float 浮点数

x = 3.14

IEEE 754 双精度

📝str 字符串

x = "hello"

不可变，Unicode

✅bool 布尔值

x = True

True / False

📋list 列表

x = [1, 2, 3]

可变，有序

📦tuple 元组

x = (1, 2, 3)

不可变，有序

🗂️dict 字典

x = {"k": "v"}

键值对，可变

🎯set 集合

x = {1, 2, 3}

无序，去重

1.2 类型操作与转换

python 
# 基本类型赋值
name = "Python"
version = 3.12
is_awesome = True
features = ["简洁", "易学", "强大"]
coordinates = (39.9, 116.4)
config = {"debug": True, "port": 8080}
unique_tags = {"python", "java", "python"}

print(type(name))         # 
print(type(version))      # 
print(unique_tags)          # 去重: {'python', 'java'}

# 类型转换
num_str = str(42)            # "42"
num_int = int("42")          # 42
num_float = float("3.14")    # 3.14
bool_val = bool(0)           # False (0/空为False)
list_from = list("abc")      # ['a', 'b', 'c']

print(type(num_str), num_str)
输出
<class 'str'> <class 'float'> {'python', 'java'} <class 'str'> 42
⚠️
常见陷阱：is 比较的是内存地址（identity），== 比较的是值（equality）。小整数（-5~256）有缓存池，a = 256; b = 256; a is b 为 True，但 a = 257; b = 257; a is b 可能为 False。永远用 == 比较值。

2条件判断（if / elif / else）

Python 用缩进（4 空格）表示代码块，没有花括号。条件表达式不需要括号，但行尾必须有冒号。

python 
score = 85

if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")

# 条件嵌套
age = 20
has_id = True
if age >= 18:
    if has_id:
        print("允许进入")
    else:
        print("需要证件")
else:
    print("未成年禁止进入")

# 多条件组合: and / or / not
is_weekend = True
is_sunny = False
if is_weekend and is_sunny:
    print("去郊游")
elif is_weekend and not is_sunny:
    print("在家看书")
else:
    print("上班")
输出
良好 允许进入 在家看书
💡
真值表：以下值在布尔上下文中为 False：None、False、0、0.0、""（空字符串）、[]（空列表）、{}（空字典）、()（空元组）、set()（空集合）。其余所有值都为 True。利用这一点可以简化：if my_list: 比 if len(my_list) > 0: 更 Pythonic。

3循环结构（for / while / else）

Python 的 for 是遍历循环（迭代器协议），不是 C 语言的计数循环。while 是条件循环。两者都有独特的 else 子句。

3.1 for 循环

python 
# 遍历列表
for fruit in ["苹果", "香蕉", "橘子"]:
    print(fruit)

# range() 生成整数序列
for i in range(3):       # 0, 1, 2
    print(f"第 {i} 次")

# range(start, stop, step)
for i in range(10, 0, -2):  # 10, 8, 6, 4, 2
    print(i, end=" ")
print()

# 遍历字典
student = {"name": "小明", "age": 18, "grade": "A"}
for key, value in student.items():
    print(f"{key}: {value}")
输出
苹果 香蕉 橘子 第 0 次 第 1 次 第 2 次 10 8 6 4 2 name: 小明 age: 18 grade: A

3.2 while 循环与 else 子句

python 
# while 基本用法
count = 0
while count < 3:
    print(f"count = {count}")
    count += 1

# for-else: 循环正常结束(未被break)时执行else
for i in range(5):
    if i == 10:  # 永远不会触发
        break
else:
    print("循环正常结束，没有break")

# 实用场景: 查找素数
for n in range(2, 10):
    for x in range(2, n):
        if n % x == 0:
            print(f"{n} = {x} * {n//x}")
            break
    else:
        print(f"{n} 是素数")
输出
count = 0 count = 1 count = 2 循环正常结束，没有break 2 是素数 3 是素数 4 = 2 * 2 5 是素数 6 = 2 * 3 7 是素数 8 = 2 * 4 9 = 3 * 3
🔑
for-else 的妙用：else 子句在循环未被 break 打断时执行。这非常适合「查找」场景：找到就 break，没找到就走 else。比用一个额外的 flag 变量更优雅。

4函数定义（def / 参数 / 返回值）

Python 函数灵活强大：支持默认参数、关键字参数、可变参数（*args / **kwargs）、仅关键字参数、类型注解等。

4.1 参数类型全解

python 
# 1. 位置参数
def greet(name):
    return f"Hello, {name}!"

# 2. 默认参数
def greet2(name, greeting="你好"):
    return f"{greeting}, {name}!"

# 3. 可变位置参数 *args
def sum_all(*args):
    return sum(args)

# 4. 可变关键字参数 **kwargs
def print_info(**kwargs):
    for k, v in kwargs.items():
        print(f"  {k} = {v}")

# 5. 仅关键字参数 (在 * 之后)
def create_user(name, *, age, email):
    return {"name": name, "age": age, "email": email}

# 调用示例
print(greet("Python"))
print(greet2("小明"))
print(greet2("小明", greeting="Hi"))
print(f"求和: {sum_all(1, 2, 3, 4, 5)}")
print("用户信息:")
print_info(name="小明", age=18, city="北京")
print(create_user("小红", age=20, email="a@b.com"))
输出
Hello, Python! 你好, 小明! Hi, 小明! 求和: 15 用户信息: name = 小明 age = 18 city = 北京 {'name': '小红', 'age': 20, 'email': 'a@b.com'}

4.2 类型注解与多返回值

python 
from typing import Optional

# 类型注解 (Python 3.6+，不影响运行但提升可读性)
def divide(a: float, b: float) -> Optional[float]:
    """两数相除，除数为0返回None"""
    if b == 0:
        return None
    return a / b

# 多返回值 (本质是返回tuple)
def min_max(numbers: list) -> tuple:
    return min(numbers), max(numbers)

# 解包接收多返回值
lo, hi = min_max([3, 1, 4, 1, 5, 9, 2, 6])
print(f"最小: {lo}, 最大: {hi}")
print(f"10/3 = {divide(10, 3):.4f}")
print(f"10/0 = {divide(10, 0)}")
输出
最小: 1, 最大: 9 10/3 = 3.3333 10/0 = None
⚠️
可变默认参数陷阱：默认参数在函数定义时只求值一次！def f(items=[]): 中 [] 会在所有调用间共享。
正确做法：def f(items=None): if items is None: items = []

5异常处理（try / except / else / finally）

Python 异常处理四段式：try 试运行 → except 捕获异常 → else 无异常时执行 → finally 必定执行。

python 
def safe_divide(a, b):
    """安全除法：处理各种异常情况"""
    try:
        result = a / b
    except ZeroDivisionError:
        print("错误: 除数不能为零")
        return None
    except TypeError as e:
        print(f"类型错误: {e}")
        return None
    else:
        print(f"计算成功: {a} / {b} = {result}")
        return result
    finally:
        print("--- 计算结束 ---")

# 测试各种情况
safe_divide(10, 3)
safe_divide(10, 0)
safe_divide("10", 3)

# 主动抛出异常
def check_age(age):
    if age < 0:
        raise ValueError(f"年龄不能为负数: {age}")
    return f"年龄合法: {age}"

try:
    check_age(-5)
except ValueError as e:
    print(f"捕获异常: {e}")
输出
计算成功: 10 / 3 = 3.3333333333333335 --- 计算结束 --- 错误: 除数不能为零 --- 计算结束 --- 类型错误: unsupported operand type(s) for /: 'str' and 'int' --- 计算结束 --- 捕获异常: 年龄不能为负数: -5
✅
四段式最佳实践：else 块放「成功后逻辑」，让 try 块只包含可能出错的代码，避免意外捕获 else 中的异常。finally 用于资源清理（关闭文件/连接），即使 return 或 raise 也会执行。
