1 Series —— 一维数据结构

Series 是 Pandas 中最基本的数据结构，类似一维数组但带有索引标签。你可以把它理解为 Excel 中的一列数据。

1.1 创建 Series 基础

Series 可以从列表、字典、NumPy 数组等多种数据源创建。每个 Series 包含两部分：数据（values）和索引（index）。

📍 业务场景

假设你在分析一家科技公司的员工薪资数据，需要存储每个部门的平均月薪。

Python · series_create.py
import pandas as pd

# ── 方式1：从列表创建（自动生成整数索引）──
salaries = pd.Series([18000, 22000, 15000, 25000])
print(salaries)

# ── 方式2：指定索引标签 ──
dept_salary = pd.Series(
    [18000, 22000, 15000, 25000],
    index=['前端', '后端', '测试', '算法']
)
print(dept_salary)

# ── 方式3：从字典创建（键自动成为索引）──
salary_dict = pd.Series({
    '前端': 18000,
    '后端': 22000,
    '测试': 15000,
    '算法': 25000
})
print(salary_dict)
▶ 运行结果
0 18000 1 22000 2 15000 3 25000 dtype: int64 前端 18000 后端 22000 测试 15000 算法 25000 dtype: int64 前端 18000 后端 22000 测试 15000 算法 25000 dtype: int64
📝
注意输出中的 dtype: int64，Pandas 会自动推断数据类型。如果混合了字符串和数字，会退化为 object 类型。

1.2 Series 属性与基本操作

获取 Series 的索引、值、数据类型，以及通过标签/位置访问元素。

Python · series_ops.py
import pandas as pd

s = pd.Series([18000, 22000, 15000, 25000],
          index=['前端', '后端', '测试', '算法'])

# ── 查看属性 ──
print("索引:", s.index.tolist())      # ['前端', '后端', '测试', '算法']
print("值:", s.values)               # [18000 22000 15000 25000]
print("数据类型:", s.dtype)           # int64
print("元素个数:", s.size)             # 4

# ── 通过标签访问（推荐）──
print("\n后端薪资:", s['后端'])          # 22000

# ── 通过位置访问（iloc）──
print("第二个元素:", s.iloc[1])        # 22000

# ── 切片访问 ──
print("\n前两个部门:")
print(s[:'后端'])  # 标签切片包含两端

# ── 向量化运算（每个元素都加2000）──
print("\n加薪后:")
print(s + 2000)
▶ 运行结果

⚠️
loc vs iloc：s.loc['后端'] 按标签访问，s.iloc[1] 按位置访问。当索引是整数时两者容易混淆，务必注意区分。

3 数据筛选与过滤

数据筛选是数据分析中最频繁的操作。Pandas 提供了强大的布尔索引和条件过滤能力，让你像写 SQL WHERE 子句一样筛选数据。

3.1 布尔索引：条件筛选 高频

📍 业务场景

电商平台运营需要从订单数据中筛选出金额大于 5000 的高价值订单，用于 VIP 客户分析。

Python · df_filter.py
import pandas as pd

df = pd.DataFrame({
    '订单ID': ['ORD001', 'ORD002', 'ORD003', 'ORD004', 'ORD005', 'ORD006'],
    '客户名': ['张三', '李四', '王五', '赵六', '钱七', '张三'],
    '商品': ['iPhone 15', 'MacBook Pro', 'AirPods Pro', 'iPad Air', 'MacBook Pro', 'Apple Watch'],
    '金额': [7999, 18999, 1999, 4799, 18999, 2999],
    '城市': ['北京', '上海', '北京', '深圳', '上海', '广州'],
})

# ── 单条件筛选：金额 > 5000 ──
high_value = df[df['金额'] > 5000]
print("=== 金额 > 5000 的高价值订单 ===")
print(high_value)

# ── 多条件筛选：金额 > 5000 且城市为北京 ──
# 注意：& 表示 AND，| 表示 OR，每个条件必须用括号包裹
result = df[(df['金额'] > 5000) & (df['城市'] == '北京')]
print("\n=== 金额>5000 且 北京 ===")
print(result)

# ── OR 条件：iPhone 或 MacBook ──
apple_products = df[(df['商品'] == 'iPhone 15') | (df['商品'] == 'MacBook Pro')]
print("\n=== iPhone 或 MacBook ===")
print(apple_products)

# ── isin：多值匹配（等价于 SQL IN）──
city_filter = df[df['城市'].isin(['北京', '上海'])]
print("\n=== 北京或上海的订单 ===")
print(city_filter)

# ── 字符串包含（等价于 SQL LIKE）──
mac_orders = df[df['商品'].str.contains('Mac')]
print("\n=== 商品名包含Mac ===")
print(mac_orders)

括号必须加：多条件组合时每个条件必须用括号包裹，否则 Python 会因运算符优先级报错。& 不是 and，| 不是 or——它们是按位运算符，用于元素级比较。

3.2 query() 方法：SQL 风格筛选

当条件复杂时，query() 方法让代码更易读，语法接近 SQL WHERE 子句。

Python · df_query.py
import pandas as pd

df = pd.DataFrame({
    '商品': ['iPhone 15', 'MacBook Pro', 'AirPods Pro', 'iPad Air', 'MacBook Pro'],
    '金额': [7999, 18999, 1999, 4799, 18999],
    '城市': ['北京', '上海', '北京', '深圳', '上海'],
})

# ── 等价于 df[(df['金额'] > 5000) & (df['城市'] == '北京')] ──
result = df.query("金额 > 5000 & 城市 == '北京'")
print("=== query 筛选 ===")
print(result)

# ── 使用 in（等价于 isin）──
result2 = df.query("城市 in ['北京', '上海']")
print("\n=== query in ===")
print(result2)

# ── 引用外部变量用 @ ──
min_amount = 5000
result3 = df.query("金额 > @min_amount")
print("\n=== 引用外部变量 ===")
print(result3)

4 分组聚合（GroupBy）

分组聚合是数据分析的灵魂操作，等价于 SQL 的 GROUP BY。Pandas 的 groupby 遵循 "Split-Apply-Combine"（分割-应用-合并）模式。

🔀
Split 分割
按某列的值
将数据分成多组
⚙️
Apply 应用
对每组独立
应用聚合函数
🔄
Combine 合并
将各组结果
合并为一张表

4.1 基本分组聚合 核心

📍 业务场景

运营需要按城市统计每个城市的订单总金额、平均金额、订单数量，用于评估各城市销售表现。

Python · df_groupby.py
import pandas as pd

df = pd.DataFrame({
    '城市': ['北京', '上海', '北京', '深圳', '上海', '北京', '深圳', '上海'],
    '商品': ['iPhone', 'MacBook', 'AirPods', 'iPad', 'MacBook', 'iPhone', 'iPad', 'AirPods'],
    '金额': [7999, 18999, 1999, 4799, 18999, 7999, 4799, 1999],
    '数量': [1, 1, 2, 1, 2, 1, 3, 1],
})

# ── 按城市分组，计算金额总和 ──
city_total = df.groupby('城市')['金额'].sum()
print("=== 各城市总金额 ===")
print(city_total)

# ── 按城市分组，多个聚合函数 ──
city_stats = df.groupby('城市')['金额'].agg(['sum', 'mean', 'count', 'max'])
print("\n=== 各城市金额统计 ===")
print(city_stats)

# ── 不同列用不同聚合函数 ──
multi_agg = df.groupby('城市').agg({
    '金额': ['sum', 'mean'],
    '数量': ['sum', 'max'],
})
print("\n=== 多列不同聚合 ===")
print(multi_agg)

# ── 自定义聚合函数名 ──
named_agg = df.groupby('城市').agg(
    总金额=('金额', 'sum'),
    平均金额=('金额', 'mean'),
    总数量=('数量', 'sum'),
    订单数=('金额', 'count'),
)
print("\n=== 命名聚合 ===")
print(named_agg)

3.2 多级分组与 transform

Python · df_groupby2.py
import pandas as pd

df = pd.DataFrame({
    '城市': ['北京', '上海', '北京', '深圳', '上海', '北京'],
    '商品': ['iPhone', 'MacBook', 'AirPods', 'iPad', 'MacBook', 'iPhone'],
    '金额': [7999, 18999, 1999, 4799, 18999, 7999],
})

# ── 多列分组：先按城市，再按商品 ──
multi_group = df.groupby(['城市', '商品'])['金额'].sum()
print("=== 多级分组 ===")
print(multi_group)

# ── reset_index：把索引变回普通列 ──
flat = multi_group.reset_index()
print("\n=== reset_index 后 ===")
print(flat)

# ── transform：保持原形状，把聚合结果广播回每一行 ──
# 场景：计算每笔订单金额占该城市总金额的比例
df['城市总额'] = df.groupby('城市')['金额'].transform('sum')
df['占比'] = (df['金额'] / df['城市总额'] * 100).round(2)
print("\n=== transform 广播 ===")
print(df)

# ── 排序：按金额降序 ──
print("\n=== 按金额降序 ===")
print(df.sort_values('金额', ascending=False))

transform vs agg：agg 会把每组压缩成一行，transform 则把聚合结果广播回原始每一行，保持 DataFrame 形状不变。当你需要在原数据上添加"组内统计"列时，用 transform。

5.2 填充缺失值（fillna）

根据业务场景选择不同的填充策略：固定值、均值、中位数、前向/后向填充。

Python · df_fillna.py
import pandas as pd
import numpy as np

df = pd.DataFrame({
    '姓名': ['张三', '李四', '王五', '赵六', '钱七'],
    '年龄': [25, np.nan, 30, 28, np.nan],
    '薪资': [15000, 20000, np.nan, np.nan, 22000],
})

# ── 策略1：用固定值填充 ──
df_fixed = df.fillna({'年龄': 0, '薪资': 0})
print("=== 固定值填充 ===")
print(df_fixed)

# ── 策略2：用均值填充（推荐）──
age_mean = df['年龄'].mean()  # 27.67
salary_mean = df['薪资'].mean()  # 19000.0
df_mean = df.fillna({'年龄': age_mean, '薪资': salary_mean})
print("\n=== 均值填充 ===")
print(df_mean)

# ── 策略3：用中位数填充（抗异常值）──
df_median = df.fillna({
    '年龄': df['年龄'].median(),
    '薪资': df['薪资'].median(),
})
print("\n=== 中位数填充 ===")
print(df_median)

# ── 策略4：前向填充（用上一行的值）──
# 常用于时间序列数据
df_ffill = df.fillna(method='ffill')
print("\n=== 前向填充(ffill) ===")
print(df_ffill)

# ── 策略5：线性插值（适合时间序列）──
df_interp = df.interpolate(method='linear')
print("\n=== 线性插值 ===")
print(df_interp)

5.3 删除缺失值（dropna）

当缺失值比例较高或填充不合理时，直接删除是更干净的选择。

Python · df_dropna.py
import pandas as pd
import numpy as np

df = pd.DataFrame({
    '姓名': ['张三', '李四', '王五', '赵六', '钱七'],
    '年龄': [25, np.nan, 30, 28, np.nan],
    '薪资': [15000, 20000, np.nan, np.nan, 22000],
})

# ── 删除任何含缺失值的行 ──
df_drop_any = df.dropna()
print("=== dropna() 删除所有含NaN的行 ===")
print(df_drop_any)

# ── 仅删除全部为NaN的行 ──
# df_drop_all = df.dropna(how='all')

# ── 指定列的缺失才删除 ──
df_drop_subset = df.dropna(subset=['年龄'])
print("\n=== 仅删除年龄列缺失的行 ===")
print(df_drop_subset)

# ── 设置阈值：非空值少于2个的行才删除 ──
df_thresh = df.dropna(thresh=2)
print("\n=== 非空值<2才删除 ===")
print(df_thresh)
