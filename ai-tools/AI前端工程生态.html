汇集当前全球最顶尖的开源项目，打通设计蓝图、代码骨架、AI 指令集到视觉自动化的完整软件生命周期链路。

探索工具链

1. 基础设施层

设计资产与代码底座

Penpot

开源 Figma 替代品。原生支持 SVG 与 CSS Flex/Grid。设计师的布局即是真实的 DOM 结构。

Pencil (pencil.dev)

商业级 AI 矢量画布。内置 MCP Server，设计文件以代码格式存在仓库中，在 IDE 内实现真正的双向同步。

OpenPencil

开源 AI 原生设计工具。首创“并发 Agent 团队”协同绘制 UI，支持一键将自然语言转化为 React 代码。

shadcn/ui

“不是组件库的组件库”。源码级组件，零封装。配合 MCP 与 AI 工具拼装界面的最佳“乐高积木”。

2. 指挥与生成层

系统规范与审美直觉

Open Design (nexu-io/open-design)

自带独立沙盒的 AI 自动化设计中枢（OS）。将底层编码助手（如 Claude Code）与顶级商业设计系统桥接，实现“选配置即生成”。

taste-skill

现象级“反平庸”架构。赋予 AI 顶级品味，拒绝大杂烩，内置可调的排版、动效与布局差异度等多维拨盘。

anthropic/design

官方原生指令，主打“拒绝 AI 塑料感”。强制大模型摒弃千篇一律的字体与紫色渐变，要求极致的视觉方向。

ui-ux-pro-max

全能型商业级设计指令集。强制 AI 遵循现代排版缩放、暗黑模式、微交互与无障碍规范。

nothing-design

极端风格化指令集。将极客黑白对比、点阵与等宽字体规则注入 AI，一键生成机械工业风界面。

3. 交互与动效层

注入灵魂的视觉魔法

Framer Motion

React 生态下最强大的动画库。通过极简的声明式 API，让 AI 轻松生成物理驱动微交互与丝滑转场。

Magic UI / Aceternity

专为现代 Web 和 AI 产品打造的高颜值开源动效库。一键引入流光边框、跟随光标磁吸等顶级大厂特效。

4. 审查与验证层

像素警察与自动化测试

refactoring-ui-plugin

将《Refactoring UI》的黄金法则化为审查插件，对杂乱代码进行自动化“视觉层级”与“留白”重构。

Midscene.js

基于视觉大模型的 UI 自动化框架。抛弃 DOM 查询，让 AI 直接“看屏幕”执行点击和端到端测试。

Case Studies

顶级 AI 产品的设计逆向拆解

用前文提到的开源设计理念，来审视目前最火的 AI 生产力工具是如何做设计的。

soft-skill

Claude Web 端

分析：极其微弱的暖麻色背景 (#FDFBF7)。抛弃了所有的生硬边框，依靠经典的衬线体排版来营造安静、克制、昂贵的“学术阅读”氛围。

nothing-design

Claude Code 终端

分析：纯正黑底白字，强制等宽字体。去除渐变和圆角，依靠 ASCII 字符和鲜明的高亮色（终端绿/黄）区分层级，将工业极客美学推向极致。

ui-ux-pro-max

Cursor 桌面端

分析：高密信息降噪教科书。杜绝乱加边框，用极微小的背景色差划分区域。极其克制的主题色只在“语法高亮”和“生成按钮”上出现。

💡 附录：小白 UI 极简设计法则

1. 黄金结构与留白 摒弃紧凑布局。卡片内部的 Padding 至少保持 16px-24px，让元素有充足的“呼吸感”。
2. 70-20-10 克制色彩原则 70% 浅色/深色背景大面铺底，20% 品牌色仅用于按钮和激活态，10% 辅助次级文本色。
3. 严控字体层级 整个页面最多使用 3 种字号（如 24px 加粗主标、14px 常规正文、12px 弱化备注）。保持左对齐。
4. 请 AI 充当像素警察 骨架拼装完成后，必须给 AI 发送 `refactoring-ui` 指令，强制其检查并添加 hover 等微交互特效。
